# 🛍️ Alma Lectora — E-Commerce & Catálogo Literario

Alma Lectora es un espacio y tienda premium diseñada especialmente para la comunidad de lectoras en Salta y toda la Argentina. Nuestro propósito es conectar a los apasionados de los libros con sus próximas lecturas favoritas y una variedad de productos complementarios que hacen de cada sesión de lectura un momento mágico, ofreciendo siempre una experiencia de compra fluida, cálida y personalizada.

## Nuestro Catálogo Comercial

En la plataforma encontrarás una selección curada con especial dedicación:
* 📚 **Libros**: Títulos destacados, novedades editoriales y los más vendidos del mercado.
* 🕯️ **Velas Aromáticas Literarias**: Diseñadas especialmente para ambientar y complementar tus sesiones de lectura.
* 🔖 **Separadores Premium**: Hechos a mano para acompañar cada historia y cuidar tus libros de forma única.
* 🧩 **Impresiones y Merchandising 3D**: Diseños exclusivos para fanáticos del mundo literario.
* 🎁 **Combos Promocionales**: Paquetes mixtos ideales con ofertas únicas para hacer un regalo especial o darse un gusto.

## Una Experiencia de Compra Directa y Humana

Operamos bajo un enfoque *Mobile-First* pensado en la comodidad de nuestras usuarias. El proceso es sumamente interactivo y sencillo: armas tu carrito de compras directamente en la web y, al finalizar, el sistema genera una orden automatizada que se envía a WhatsApp. De esta manera, el pago y los detalles del envío se coordinan de forma directa, humana, segura y personalizada con nosotras.

### Resumen del Stack Tecnológico

* **Frontend**: React 19, Tailwind CSS v4, TanStack Query v5, Zustand.
* **Backend**: Node.js, Express 5, TypeScript, Prisma ORM, PostgreSQL.
* **Infraestructura**: Despliegue híbrido en Vercel y persistencia en Supabase.
